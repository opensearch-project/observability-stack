`OpenSearchTraceRetriever` queries stored GenAI traces from OpenSearch. Use it to build evaluation pipelines that retrieve past agent runs, score them, and upload results.

## Installation

Trace retrieval requires the `opensearch` extra:

```bash
pip install "opensearch-genai-observability-sdk-py[opensearch]"
```

## Quick start

```python
from opensearch_genai_observability_sdk_py import OpenSearchTraceRetriever

retriever = OpenSearchTraceRetriever(
    host="https://localhost:9200",
    auth=("admin", "admin"),
    verify_certs=False,
)

# List recent root spans
roots = retriever.list_root_spans(max_results=10)
for span in roots:
    print(f"{span.trace_id} | {span.agent_name} | {span.name}")

# Get full trace with all spans
session = retriever.get_traces(roots[0].trace_id)
for t in session.traces:
    for span in t.spans:
        print(f"  {span.name} ({span.operation_name})")
```

## `OpenSearchTraceRetriever`

### Constructor

```python
OpenSearchTraceRetriever(
    host: str = "https://localhost:9200",
    index: str = "otel-v1-apm-span-*",
    auth: tuple[str, str] | RequestsAWSV4SignerAuth | None = None,
    verify_certs: bool = True,
)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `host` | `str` | `"https://localhost:9200"` | OpenSearch endpoint URL. |
| `index` | `str` | `"otel-v1-apm-span-*"` | Index pattern to query. |
| `auth` | `tuple` or `RequestsAWSV4SignerAuth` | | Basic auth tuple `("user", "pass")` or AWS SigV4 signer. |
| `verify_certs` | `bool` | `True` | Verify SSL certificates. |

### `get_traces()`

Retrieves all spans for a session or trace.

```python
session = retriever.get_traces("session-123", max_spans=10000)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `identifier` | `str` | | Session ID (`gen_ai.conversation.id`) or trace ID. Queries by session first, falls back to trace ID. |
| `max_spans` | `int` | `10000` | Maximum number of spans to retrieve. |

Returns a `SessionRecord`.

### `list_root_spans()`

Lists recent root spans (spans without a parent).

```python
from datetime import datetime, timedelta

roots = retriever.list_root_spans(
    services=["my-agent"],
    since=datetime.utcnow() - timedelta(hours=1),
    max_results=50,
)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `services` | `list[str]` | | Filter by service name. |
| `since` | `datetime` | 15 minutes ago | Only return spans after this time. |
| `max_results` | `int` | `50` | Maximum number of root spans to return. |

Returns a `list[SpanRecord]`.

### `find_evaluated_trace_ids()`

Returns the subset of trace IDs that already have an evaluation span.

```python
trace_ids = [span.trace_id for span in roots]
already_scored = retriever.find_evaluated_trace_ids(trace_ids)
needs_scoring = [tid for tid in trace_ids if tid not in already_scored]
```

| Parameter | Type | Description |
|---|---|---|
| `trace_ids` | `list[str]` | List of trace IDs to check. |

Returns a `set[str]` of trace IDs that have evaluation spans.

## Data models

### `SessionRecord`

```python
@dataclass
class SessionRecord:
    session_id: str
    traces: list[TraceRecord]
    truncated: bool
```

### `TraceRecord`

```python
@dataclass
class TraceRecord:
    trace_id: str
    spans: list[SpanRecord]
```

### `SpanRecord`

```python
@dataclass
class SpanRecord:
    trace_id: str
    span_id: str
    parent_span_id: str | None
    name: str
    start_time: str
    end_time: str
    operation_name: str | None          # gen_ai.operation.name
    agent_name: str | None              # gen_ai.agent.name
    model: str | None                   # gen_ai.request.model
    input_messages: list[Message]       # from events or attributes
    output_messages: list[Message]      # from events or attributes
    tool_name: str | None               # gen_ai.tool.name
    tool_call_arguments: str | None     # gen_ai.tool.call.arguments
    tool_call_result: str | None        # gen_ai.tool.call.result
    input_tokens: int | None            # gen_ai.usage.input_tokens
    output_tokens: int | None           # gen_ai.usage.output_tokens
    raw: dict                           # original OpenSearch document
```

### `Message`

```python
@dataclass
class Message:
    role: str
    content: str
```

## Evaluation pipeline example

A complete workflow: retrieve traces, check which need scoring, evaluate, and write scores back.

```python
from opensearch_genai_observability_sdk_py import (
    register, score, OpenSearchTraceRetriever,
)

register(service_name="eval-pipeline")

retriever = OpenSearchTraceRetriever(
    host="https://localhost:9200",
    auth=("admin", "admin"),
    verify_certs=False,
)

# 1. Find recent agent traces
roots = retriever.list_root_spans(services=["my-agent"], max_results=20)

# 2. Filter out already-evaluated traces
trace_ids = [s.trace_id for s in roots]
evaluated = retriever.find_evaluated_trace_ids(trace_ids)
to_evaluate = [s for s in roots if s.trace_id not in evaluated]

# 3. Retrieve full traces and score them
for root in to_evaluate:
    session = retriever.get_traces(root.trace_id)
    for t in session.traces:
        # Run your evaluation logic on the trace data
        relevance = compute_relevance(t.spans)
        score(
            name="relevance",
            value=relevance,
            trace_id=root.trace_id,
            explanation="Automated relevance check",
        )
```

## AWS authentication

For AWS-hosted OpenSearch, use `RequestsAWSV4SignerAuth` from `opensearch-py`:

```python
from opensearchpy import RequestsAWSV4SignerAuth
import boto3

credentials = boto3.Session().get_credentials()
auth = RequestsAWSV4SignerAuth(credentials, "us-east-1", "es")

retriever = OpenSearchTraceRetriever(
    host="https://search-my-domain.us-east-1.es.amazonaws.com",
    auth=auth,
)
```
